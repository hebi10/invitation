# 관리자 대시보드 탭 배치

## 현재 기준

- 상단 첫 번째 탭 바는 `관리 섹션` 영역으로 유지합니다.
- `이벤트 운영 > 모바일 청첩장` 화면일 때만 `페이지 유형 탭`을 같은 상단 탭 바 내부에 함께 노출합니다.
- `세부 관리 탭` 바는 기존처럼 별도 줄에 유지합니다.

## 현재 동작

1. `관리 섹션`
   - `고객 관리`, `이벤트 운영` 전환을 담당합니다.
2. `페이지 유형 탭`
   - `이벤트 운영` 섹션의 `모바일 청첩장` 탭이 활성화된 경우에만 상단 `관리 섹션` 탭 바 안에 함께 나타납니다.
   - 현재 실제 목록이 준비된 유형은 `청첩장`이며, 나머지 유형은 TODO 안내 패널을 보여줍니다.
3. `세부 관리 탭`
   - `고객 계정`, `모바일 청첩장`, `이미지`, `방명록` 같은 실제 작업 탭은 기존 별도 탭 바에서 유지합니다.
